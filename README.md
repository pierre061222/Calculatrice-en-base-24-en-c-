# Calculatrice-en-base-24-en-c-

Cette calculatrice permet de faire des opérations en base 24

# Fonctionnalités

1.Addition en base 24
2.soustraction en base 24
3.Multiplication en base 24
4.Division en base 24  
5.Conversion base 10 en base 24

# Comment utiliser le programme

1. Lancez l’exécutable ou compilez le programme.
2. Choisissez l’opération :
   -  Addition  
   - Soustraction  
   - Multiplication  
   - Division  
3. Entrez les deux nombres en base 24.
4. Le programme affiche :
   -  le calcul en base 10  
   -  le résultat final en base 24
